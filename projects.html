SMART-DOOR

A secure, automated door lock using RFID technology. Authorized users scan their RFID tag/card with an RC522 reader, and an Arduino/ESP8266 validates access. If approved, a servo or solenoid lock opens the door, with LED/buzzer feedback. Optional Wi-Fi (NodeMCU) enables remote logging of entries.

OBSTACLE AVOIDANCE ROBOT CAR

A smart autonomous vehicle built using Arduino (C++) that navigates environments independently. Equipped with ultrasonic (HC-SR04) and PIR sensors, it detects obstacles and motion in real-time. The car avoids collisions using programmed logic, adjusting its path with DC motors (L298N driver). Ideal for robotics beginners, this project demonstrates sensor integration, real-time decision-making, and motor control.
